# Bharatversity AI/ML Internship Project - Linear Regression

This project was completed as part of my AI/ML internship with Bharatversity. It focuses on applying Linear Regression to analyze retail pricing data and predict profit or pricing-related outcomes using Python and machine learning.

## Project Overview

The goal of this project is to understand how different retail factors such as product price, competitor pricing, quantity, cost, and sales influence business profit and pricing decisions.

Using data analysis and machine learning, this project builds a Linear Regression model to identify patterns and generate useful business insights.

## Objectives

- Clean and preprocess retail pricing data
- Perform exploratory data analysis
- Understand relationships between pricing, cost, quantity, and profit
- Build a Linear Regression model
- Evaluate model performance
- Generate business insights from the data
